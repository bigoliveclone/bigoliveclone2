Important Notice

This package is a frontend template only. It does not include a backend, live streaming server, payment gateway, database, admin panel, or production authentication service.

Installation

npm install

Recommended runtime: Node.js 20 or newer and npm 10 or newer.

Run Development Server

npm run dev

Build for Production

npm run build

The production files will be generated in the dist/ directory.

Project Structure

Path Description
src/components/ Reusable Vue UI components.
src/pages/ Route-level pages.
src/mock/ Local mock data for streams, gifts, rankings, wallet records, and chats.
src/router/ Vue Router configuration.
src/store/ Pinia state store with localStorage demo persistence.
src/styles/ Global mobile H5 CSS.

Customize Colors

Edit the CSS variables in src/styles/main.css:

:root {
  --ink: #17201d;
  --paper: #fffaf1;
  --accent: #ff5d4d;
  --accent-2: #18b7a4;
  --dark: #111614;
}

Edit Mock Data

Open src/mock/liveData.js to change live rooms, gifts, recharge packages, ranking lists, host profiles, and chat messages.
